# This repository is for the Semester Project.

## Initializing and starting the project

- Clone the project and initialize git in your code editor.

```
 git init
```

- Install Dependencies.

```
 npm install
```

- Build the project.

```
 npm run build
```

- Start the project with live-server.

```
 npm run start
```
